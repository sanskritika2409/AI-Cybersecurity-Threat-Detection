🛡️ AI Cybersecurity Threat Detection System
📌 Overview

This project detects cyber threats using Machine Learning and analyzes network traffic data.

🎯 Features
Detects normal vs malicious traffic
Machine Learning model (Random Forest)
Streamlit dashboard
Data visualization
🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Streamlit, Matplotlib

📂 Project Structure
data/
models/
src/
main.py
app.py
requirements.txt
▶️ How to Run
pip install -r requirements.txt
python main.py
streamlit run app.py
